# cnet
计算子网，支持ipv4和ipv6
## 1. cnet ip地址 掩码或前缀 或 cnet IP地址/掩码或前缀
* IPv4
```
$ cnet 192.168.1.23 25
---------sub calc (IPv4)---------
IP 位置/可用IP     : 192.168.1.23 (C) 23 / 126
网络~广播地址      : 192.168.1.0 ~ 192.168.1.127
掩码|前缀|反掩码   : 255.255.255.128 | 25 | 0.0.0.127
```
* IPv6
```
cnet 2001::1 66
---------sub calc (IPv6)---------
IPv6地址         : 2001::1
网络~最大地址    : 2001:: ~ 2001::3fff:ffff:ffff:ffff
前缀长度         : 66
可用地址数量     : 4611686018427387902
```
## 2. cnet ip地址 掩码或前缀 分割掩码 或 cnet IP地址/掩码或前缀 分割掩码
* IPv4 分割掩码
```
$ cnet 192.168.1.23 25 27
---------sub calc (IPv4)---------
IP 位置/可用IP     : 192.168.1.23 (C) 23 / 126
网络~广播地址      : 192.168.1.0 ~ 192.168.1.127
掩码|前缀|反掩码   : 255.255.255.128 | 25 | 0.0.0.127

---------DivSubnet---------
划分 4 个 (255.255.255.224 | 27 | 0.0.0.31) 子网:
子网 1: 192.168.1.0 ~ 192.168.1.31
子网 2: 192.168.1.32 ~ 192.168.1.63
子网 3: 192.168.1.64 ~ 192.168.1.95
子网 4: 192.168.1.96 ~ 192.168.1.127
```


* IPv6 分割掩码
```
$ cnet 2001::1 66 69
---------sub calc (IPv6)---------
IPv6地址         : 2001::1
网络~最大地址    : 2001:: ~ 2001::3fff:ffff:ffff:ffff
前缀长度         : 66
可用地址数量     : 4611686018427387902

---------DivSubnet---------
划分 8 个 69 子网:
子网 1: 2001:: ~ 2001::7ff:ffff:ffff:ffff
子网 2: 2001::800:0:0:0 ~ 2001::fff:ffff:ffff:ffff
子网 3: 2001::1000:0:0:0 ~ 2001::17ff:ffff:ffff:ffff
子网 6: 2001::2800:0:0:0 ~ 2001::2fff:ffff:ffff:ffff
子网 7: 2001::3000:0:0:0 ~ 2001::37ff:ffff:ffff:ffff
子网 8: 2001::3800:0:0:0 ~ 2001::3fff:ffff:ffff:ffff
```
